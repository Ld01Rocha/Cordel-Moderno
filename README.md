# Cordel-Moderno
O Cordel Moderno é um site responsivo que apresenta o poema de Milton Duarte sobre o impacto da tecnologia. Desenvolvido com HTML5 e CSS3, o projeto utiliza o efeito parallax e tipografia temática para unir a estética da literatura de cordel ao design web moderno, focando em uma experiência de leitura imersiva e estilização avançada.
